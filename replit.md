# Personal Portfolio Website

## Overview
A visually impressive personal portfolio website built with pure HTML5 and CSS3, featuring dynamic animations and smooth color transitions. The design showcases professional information including personal details, skills, projects, work experience, and contact information.

## Features
- **Dynamic Animations**: Smooth color transitions flowing between vibrant and subdued states
- **Animated Gradients**: Beautiful gradient backgrounds that shift and flow continuously
- **Pulsating Elements**: Pulse rings, floating shapes, and animated icons
- **Hover Effects**: Interactive hover transitions on buttons, cards, and navigation
- **Responsive Design**: Fully responsive layout optimized for desktop, tablet, and mobile
- **Pure CSS**: No JavaScript or external libraries - all animations done with CSS keyframes

## Sections
1. **Hero Section**: Animated gradient background with floating shapes and gradient text
2. **About Section**: Rotating profile placeholder with pulse effect and animated statistics
3. **Skills Section**: Animated progress bars with shimmer effects and color-shifting badges
4. **Projects Section**: Card-based gallery with scaling animations and gradient overlays
5. **Experience Section**: Timeline layout with sliding animations and pulsating markers
6. **Contact Section**: Animated form elements with bouncing icons and hover effects

## Technologies Used
- HTML5 (semantic markup)
- CSS3 (custom properties, keyframe animations, transforms, transitions)
- CSS Grid and Flexbox for responsive layouts
